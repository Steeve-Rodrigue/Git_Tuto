## Method 2: Pre-commit Framework (Hybrid)

The pre-commit framework manages hooks via a **version-controlled config file**. The hybrid approach combines **remote hooks** (community-maintained) and **local hooks** (your custom scripts).

---

## Why Pre-commit Framework?

| Feature | Local Hooks (Part 2) | Pre-commit Framework |
|---------|---------------------|---------------------|
| Version controlled | ❌ No | ✅ Yes |
| Team sharing | ❌ Manual | ✅ Automatic |
| Updates | ❌ Manual | ✅ `pre-commit autoupdate` |
| Ready-to-use hooks | ❌ Build yourself | ✅ Thousands available |

**Hybrid = Best of both worlds:**
- Remote hooks for standard tools (Ruff, ESLint)
- Local hooks for custom project rules

## Project Structure
````
my-project/
├── .git/
│   └── hooks/
│       └── pre-commit              # Auto-generated by pre-commit
├── .venv/                          # UV virtual environment
├── .pre-commit-config.yaml         # Hook configuration (tracked)
├── pyproject.toml                  # UV + Ruff config
├── uv.lock                         # UV lock file
├── hooks/                          # Custom local hooks (tracked)
│   ├── ruff_custom.py
│   └── custom_checks.py
├── src/
│   ├── __init__.py
│   └── main.py
└── README.md
````

**Key files:**
- `.pre-commit-config.yaml` → Version controlled ✅
- `hooks/` directory → Version controlled ✅
- `.git/hooks/pre-commit` → Auto-generated, don't edit
- `pyproject.toml` → UV dependencies + Ruff config

## Installation with UV

### Step 1: Add Pre-commit with UV
````bash
# Add pre-commit as dev dependency
uv add --dev pre-commit

# Sync dependencies
uv sync
````

### Step 2: Install Git Hooks
````bash
# Activate UV environment
source .venv/bin/activate

# Install pre-commit hook
pre-commit install

# Output: pre-commit installed at .git/hooks/pre-commit
````

### Step 3: Add Ruff (if not already done)
````bash
# Add Ruff as dev dependency
uv add --dev ruff

# Sync
uv sync
````

## Hybrid Configuration

### Basic Setup

Create `.pre-commit-config.yaml` in repository root:
````yaml
# .pre-commit-config.yaml

repos:
  # -------------------------------------
  # REMOTE HOOKS: Community maintained
  # -------------------------------------
  - repo: https://github.com/astral-sh/ruff-pre-commit
    rev: v0.14.14
    hooks:
      - id: ruff-format      # Auto-format
      - id: ruff             # Lint and check
        args: ['--fix']

  - repo: https://github.com/pre-commit/pre-commit-hooks
    rev: v4.5.0
    hooks:
      - id: trailing-whitespace
      - id: end-of-file-fixer
      - id: check-yaml
      - id: detect-private-key

  # -------------------------------------
  # LOCAL HOOKS: Your custom scripts
  # -------------------------------------
  - repo: local
    hooks:
      - id: custom-checks
        name: Project Checks
        entry: python hooks/custom_checks.py
        language: system
        types: [python]
        pass_filenames: true
````

**Note**: Removed `ruff-custom` hook since remote `ruff` hook is sufficient.

## Custom Local Hooks

### Create hooks/ Directory
````bash
mkdir -p hooks
````

### Hook: Custom Project Checks
````python
#!/usr/bin/env python3
# hooks/custom_checks.py

import sys

print("=== Custom Checks ===")

files = sys.argv[1:]
errors = False

for f in files:
    content = open(f).read()
    
    # Check for print() statements
    if 'print(' in content and '/tests/' not in f:
        print(f"❌ {f}: Found print()")
        errors = True
    
    # Check for bare except
    if 'except:' in content:
        print(f"❌ {f}: Found bare except")
        errors = True

if errors:
    sys.exit(1)

print("✅ OK")
sys.exit(0)
````

### Make Executable
````bash
chmod +x hooks/custom_checks.py
````

## Usage Workflow

### First Time Setup (Project Creator)
````bash
# 1. Initialize UV project
uv init
cd my-project

# 2. Add dependencies
uv add --dev pre-commit ruff
uv sync

# 3. Create config file
touch .pre-commit-config.yaml
# (add configuration from above)

# 4. Create hooks directory
mkdir hooks
# (add custom_checks.py)

# 5. Install hooks
source .venv/bin/activate
pre-commit install

# 6. Run on all files
pre-commit run --all-files

# 7. Commit everything
git add .
git commit -m "chore: setup pre-commit hooks"
````

### Team Member Setup
````bash
# 1. Clone repository
git clone <repo-url>
cd my-project

# 2. Install dependencies with UV
uv sync

# 3. Activate environment
source .venv/bin/activate

# 4. Install hooks
pre-commit install

# Done! Hooks ready to use
````

## pyproject.toml Configuration

Complete example with UV and Ruff:
````toml
# pyproject.toml

[project]
name = "my-project"
version = "0.1.0"
description = "My project with Git hooks"
requires-python = ">=3.11"
dependencies = []

[project.optional-dependencies]
dev = [
    "pre-commit>=3.0.0",
    "ruff>=0.1.0",
]

[tool.uv]
dev-dependencies = [
    "pre-commit>=3.0.0",
    "ruff>=0.1.0",
]

[tool.ruff]
line-length = 88
target-version = "py311"

select = [
    "E",   # pycodestyle errors
    "F",   # pyflakes
    "I",   # isort
]

ignore = ["E501"]

exclude = [".git", "__pycache__", ".venv"]

[tool.ruff.format]
quote-style = "single"
````

## Commands

### Essential Commands
````bash
# Install hooks
pre-commit install

# Run manually on all files
pre-commit run --all-files

# Run specific hook
pre-commit run ruff --all-files

# Update to latest versions
pre-commit autoupdate

# Run Ruff manually (via UV)
uv run ruff check .
uv run ruff format .

# Bypass hooks (emergency only)
git commit --no-verify -m "message"
````

### UV-Specific Commands
````bash
# Add new dev dependency
uv add --dev <package>

# Run command in UV environment
uv run <command>

# Update all dependencies
uv lock --upgrade

# Sync dependencies
uv sync
````

## Comparison

| Aspect | Part 2 (Local) | Part 3 (Pre-commit + UV) |
|--------|---------------|--------------------------|
| **Location** | `.git/hooks/` | `.pre-commit-config.yaml` + `hooks/` |
| **Version control** | ❌ No | ✅ Yes |
| **Dependencies** | Manual | ✅ UV manages |
| **Team setup** | Manual copy | `uv sync` + `pre-commit install` |
| **Updates** | Manual | `pre-commit autoupdate` |
| **Remote hooks** | ❌ No | ✅ Yes (Ruff, ESLint, etc.) |
| **Best for** | Personal | Teams with UV |

## Quick Reference

### Update Everything
````bash
# Update UV dependencies
uv lock --upgrade
uv sync

# Update pre-commit hooks
pre-commit autoupdate

# Commit changes
git add pyproject.toml uv.lock .pre-commit-config.yaml
git commit -m "chore: update dependencies and hooks"
````

### Troubleshooting
````bash
# Validate pre-commit config
pre-commit validate-config

# Run verbose
pre-commit run --all-files --verbose

# Clear pre-commit cache
pre-commit clean

# Reinstall UV environment
uv sync --reinstall
````
